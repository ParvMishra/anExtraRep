About: "It is a simple HTML form that has the following entries: name, age, weight, email, and upload health report. 
        The data entered will be stored in a database."

SetUp:  

    Step 1: create a database from the "db/userDetails.sql".
     
    Step 2: Execute the index file, where users can provide their credentials and store the data in the database. 

    Step 3: If users want to view their uploaded health report, they can easily do so by going to the "check report" section and providing their email. 

    Step 4: They will then receive the PDF file.


Tech Used: HTML,CSS,Php Mysql
